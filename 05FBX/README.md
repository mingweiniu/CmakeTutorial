# 05FBX
create polygonal model

# Prerequisites  
  * CMake (3.5 or latter)  
  * FBX (2017.1)  
  * Boost (1.60 or latter)

## Command on Windows PowerShell:  
First, you need $ENV:FBX_ROOT
Second, you need add C:\Windows\Microsoft.NET\Framework\\_v4.0.30319_ to Path  
PS\\> ./build.bat

## Command on  Linux:
[user]$ bash build.sh
